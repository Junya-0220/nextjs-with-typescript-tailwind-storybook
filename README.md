## Next.js 開発サーバ

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

## Storybook 開発サーバ

https://www.learnstorybook.com/intro-to-storybook/react/en/get-started/(参考記事)

```bash
yarn storybook
```

http://localhost:6006/ にアクセスし、storybook が立ち上がることを確認する

lint を実行

```bash
yarn lint
```

fix を実行

```bash
yarn lint:fix
```

prettier を実行

```bash
yarn format
```
